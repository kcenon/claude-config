# Project Command Policies

These policies apply to all commands in this directory.

## Output Format

| Aspect | Standard |
|--------|----------|
| Default format | Markdown tables and code blocks |
| Technical content | English |
| Structured data | Tables with clear headers |

## Error Reporting

| Aspect | Standard |
|--------|----------|
| Format | Structured with severity level |
| Content | Actionable resolution steps |
| Verbosity | Concise but complete |

## Code Analysis Standards

| Aspect | Standard |
|--------|----------|
| Metrics | Use quantitative thresholds where applicable |
| Severity | Classify as Critical, High, Medium, Low |
| Suggestions | Provide concrete, actionable improvements |

## Attribution

**No AI/Claude attribution in any output.** See global `commit-settings.md` for the full policy.
